# Week 8 – Introduction to Machine Learning for IoT

## Overview
This project introduces Machine Learning techniques for IoT sensor data analysis. Three machine learning models were implemented to predict sensor values and classify system states.

## Objective
- Apply Machine Learning techniques to IoT data.
- Implement Linear Regression.
- Implement Decision Tree.
- Implement Random Forest.
- Compare model performance.

## Dataset
- Better_Dataset.csv
- Records: 2311

## Models Implemented
- Linear Regression
- Decision Tree
- Random Forest

## Results

| Model | Result |
|-------|--------|
| Linear Regression | Mean Squared Error = 1.34396e-29 |
| Decision Tree | Accuracy = 61.77% |
| Random Forest | Accuracy = 72.14% |

## Best Model
Random Forest achieved the highest classification accuracy of **72.14%**.
