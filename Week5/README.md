# Week 5 – Exploratory Data Analysis (EDA)

## Overview
This repository contains the work completed during **Week 5** of the IoT Data Analyst Internship. The objective of this task was to perform **Exploratory Data Analysis (EDA)** on an IoT sensor dataset to understand its structure, identify patterns, and visualize relationships between different variables.

## Objectives
- Load and inspect the dataset
- Analyze the dataset structure
- Generate statistical summaries
- Check for missing values
- Perform correlation analysis
- Visualize data using different plots
- Analyze temperature and humidity trends over time
- Draw meaningful insights from the dataset

## Dataset Information
- **Dataset Name:** Better_Dataset.csv
- **Total Records:** 2311
- **Total Columns:** 4

### Dataset Columns
- Date
- Temperature
- Humidity
- Verdict

## Tools and Libraries Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Loaded the dataset using Pandas
- Displayed the first five rows using `df.head()`
- Examined dataset information using `df.info()`
- Generated statistical summary using `df.describe()`
- Checked for missing values
- Created a correlation heatmap
- Generated histogram plots
- Created box plots
- Generated scatter plots
- Performed time-series analysis
- Compared temperature and humidity over time

## Results
- Successfully analyzed **2311 IoT sensor records**
- No missing values found
- Identified a moderate negative correlation between temperature and humidity
- Visualized sensor data using multiple graphs
- Gained insights into sensor behavior and environmental trends
