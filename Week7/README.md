# Week 7 – Anomaly Detection in IoT Data

## Overview
This project focuses on detecting anomalies in an IoT sensor dataset using different anomaly detection techniques.

## Objective
- Detect abnormal sensor readings
- Compare multiple anomaly detection methods
- Visualize detected anomalies

## Dataset
- **Dataset:** Better_Dataset.csv
- **Records:** 2311
- **Features:**
  - Date
  - Temperature
  - Humidity
  - Verdict

## Techniques Used
- Threshold-Based Detection
- Isolation Forest
- Local Outlier Factor (LOF)

## Tools and Libraries
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- Threshold-Based Detection: 206 anomalies
- Isolation Forest: 116 anomalies
- Local Outlier Factor (LOF): 116 anomalies

## Files Included
- Better_Dataset.csv
- Week7_Anomaly_Detection.ipynb
- Week7_Anomaly_Detection_Report.docx
- Week7_Anomaly_Detection_Report.pdf
- README.md

## Conclusion
Three anomaly detection methods were successfully implemented and compared. Machine learning-based methods such as Isolation Forest and LOF provided more intelligent anomaly detection compared to the threshold-based approach.
