# IoT Fundamentals

## What is IoT?

The Internet of Things (IoT) is a network of physical devices connected to the internet that collect, share, and analyze data.

## Components of IoT

- Sensors
- Actuators
- Connectivity
- Cloud
- Data Analytics
- Dashboard

## Sensors

Sensors collect information from the environment.

Examples:
- Temperature Sensor
- Humidity Sensor
- Motion Sensor
- Gas Sensor
- Light Sensor

## Actuators

Actuators perform actions based on commands.

Examples:
- LED
- Fan
- Relay
- Motor
- Buzzer

## Applications

- Smart Homes
- Smart Agriculture
- Healthcare
- Smart Cities
- Industrial Automation
