Week 9 – Time-Series Analysis and Forecasting
Overview

This project demonstrates Time-Series Analysis and Forecasting using an IoT sensor dataset. The notebook performs moving average analysis, trend analysis, and temperature forecasting using Linear Regression.

Objective
Analyze temporal sensor data.
Study moving averages.
Perform trend analysis.
Forecast future temperature values.
Understand time-series behavior in IoT systems.
